Setup
------------
```sh
$ git clone https://github.com/Benyamin-Rafaeli/cy
$ cd cy
```

Installation
------------
```sh
npm i
```

Run headed
----------
```sh
npm run cy:headed
```

Run headless
----------
```sh
npm run cy
```

Runner
----------
```sh
npm run cyx
```
